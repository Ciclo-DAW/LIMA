CAPÍTULO 1

Etiquetas HTML

Para comezar a crear as nosas páxinas web o único que temos que facer e ir compoñendo as diferente etiquetas HTML dun documento. Unha etiqueta HTML é un termo rodeado por un signo de menor e outro de maior.

Estructura básica dunha páxina

As páxinas web manteñen unah estrutura moi sinxera que debemos respectar para que os navegadores sexan capaces de presentala. Non podemos comezar a nos páxina cunha etiqueta de párrafo, por exemplo, senón que debemos indicar que tipo de páxina estamos xenerando, que información adicional levará e onde comeza o contido que debe ser amosado.

CAPÍTULO 2

Títulos e párrafo

A linguaxe HTML é moi coidadosa coa organización da información, polo que o primero que debemos coñecer é como estruturar o títulos e como definir os párrafos de texto.

Párrafos

As etiquetas p y /p empréganse para definir un bloque de texto que se comporta como un párrafo. Normalmente non deixaremos nunca unha porción de texto solta pola páxina web, senón que a rodearemos con esas etiquetas.

Títulos

As etiquetas desde (h1 a h6) utilízanse para definir un texto como título, indicando que é unha cabeceira que queremos destacar sobre o resto do texto. Temos 6 opcións para definir diferentes títulos de maior a menor importancia. Unha páxina web ben diseñada contará con estos encabezamentos para definir os distintos apartados do texto, cos seus diferentes niveis.
